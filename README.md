# Python Basics Repository

## Project Purpose
This repository contains basic Python concepts explained using Jupyter Notebooks.
It is designed for beginners who want to learn Python programming step by step.

## Technologies Used
- Python
- Jupyter Notebook
- Google Colab

## Topics Covered
- Python data types
- Conditional statements (if, else)
- For loops
- List methods and operations

## Files Included
- `Data_types.ipynb` – Python data types examples
- `IF_ELSE_FOR_LOOP.ipynb` – Conditional statements and loops
- `List_methods.ipynb` – List operations and methods

## How to Use
1. Clone or download this repository
2. Open notebooks using Jupyter Notebook or Google Colab
3. Run each cell to understand Python concepts
